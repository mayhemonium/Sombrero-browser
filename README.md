# Sombrero Browser 

Sombrero is a **minimal, fast, and privacy-focused web browser** built with Flutter.  
This project is an **Early Beta / Early Release**, showcasing a lightweight browser with core features ready for testing and feedback.

## Features
- **4 Search Engines**: Google (default), Brave, Yahoo, DuckDuckGo — users can switch at any time.
- **Ad & Tracker Blocking**: Built-in, efficient blocker to remove ads and protect your privacy.
- **Incognito Tabs**: Browse privately; cookies and history are not stored (note: some sites like Google may still detect existing logins).
- **Minimal & Clean UI**: Simple interface for a smooth browsing experience.
- **Open Source & Early Beta**: Transparent development, feedback welcomed!

## Known Issues
- Some lag and memory leaks (will be fixed in upcoming releases)
- Google login may occasionally fail or require multiple clicks
- Incognito mode is not yet fully isolated for all sites
